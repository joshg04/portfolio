Pokedex Web App

This web application showcases over 150 Pokémon using the Pokémon API, dynamically fetching and displaying data. Built with Svelte.js, HTML, and CSS, the app provides a user-friendly interface to explore detailed Pokémon information. The project demonstrates API integration, modern front-end development techniques, and responsive design, offering an engaging and visually appealing user experience.
